Ubuntu Kernel Team
This site is for accessing various information that is important for the Ubuntu Kernel Team. Various reports, and tools that help work with the bugs in Launchpad.
 

Reports
  The SRU Report A list of all the supported series that have packaegs currently in -proposed. This also shows all associated bugs and their status with respect to being verified.
  Package Versions A table showing all the Kernel Teams' packages and the latest version in any pocket.
  Release Tracking Bugs Every SRU kernel uploaded has an accompanying tracking bug attached to it. This is a list of all of those tracking bugs that are currently active. The information in the report shows the current state of the release in the kernel SRU cadence and how long it's been in any given state.
  Kernel CVE Bugs Currently open Kernel CVE bugs.
  Kernel CVE Matrix Currently open Kernel CVEs by topic branch and series.
  New, Last 24 Hours All bugs in any Kernel Team package that have been created in the last 24 hours.
  New, Last 30 Days All bugs in any Kernel Team package that have been created in the last 30 days.
  Kernel Team Meeting Stats Bug stats that are reported in the weekly Kernel Team IRC Meeting
  Kernel Team Meeting CVE Stats CVE stats that are reported in the weekly Kernel Team IRC Meeting
